Apa itu Fullstack Developer?

Fullstack Developer adalah seorang programmer yang mengerjakan 2 role sekaligus yaitu mengerjakan tugas frontend developer dan backend developer.

Penjelasan Sederhana

Fullstack Developer adalah orang yang bisa membuat website atau aplikasi dari depan sampai belakang. Contoh sederhana :

  • Membuat tampilan website/aplikasi yang dilihat dan digunakan oleh pengguna (Ui/Ux), Frontend
  • Mengelola server database, dan logika aplikasi (data, login, Api), Backend
  • Mengerjakan Frontend dan Backend Sekaligus ,Fullsatck

Tugas Utama Fullstack Developer

  • Membuat desain web menggunakan css atau tools mendesain web lainnya.
  • Mengelola server atau memantau server jika ada kendala.
  • Memperbaiki bug / error pada suatu codingan dan masalah lainnya.
  • Mengembangkan fitur baru di web atau aplication.
  • Bekerja sama dengan tim (desainer, backend, dll).

Tahap-Tahap Yang Kalian Harus Hadapi Kalo Mau Belajar Jadi Programmer yang Ahli

Tahap 1-Dasar (Kelas X)

    Berikut tahap awal yg wajib di pelajari :

  • Mempelajari menjadi frontend di awal agar biar bisa ahli di tahap menengah nya dalam mengurus desain Web/aplication.
  • Meningkatkan algoritma penyelesaian masalah.
  • sering-sering latihan ngoding dan ngefrontend kalo sudah di bimbing guru.
  • Sering belajar dirumah dalam mengoding HTML , CSS , JS , dan yang paling wajib adalah Python.
  • sering-sering mencari tantangan yang menantang / menyeselesaikan masalah besar.
  • Mengikuti Eskul IT di sekolah.

Yang disarankan untuk pemula adalah mengambil peran frontend dan web design.

    Berikut Materi awal yang harus di pelajari selama semester 1-2 :

  • Mendownload tools yang bernama Visual Code Studio link nya click download
  • Mempelajari html dasar - html yang hard
  • Mempelajari Css Style agar memperindah webmu
  • Mempelajari dan meningkatkan belajar javascript
  • Mempelajari dasar python
  • Belajar Mosting web di Git-hub sebagai hasil karya andalan web yang bagus menurutmu

Tahap 2-Menengah (Kelas XI)

    Berikut tahap Menengah yg wajib di pelajari :

Tahap 3-Akhir/Lulus (Kelas XII)

    Berikut tahap Akhir yg wajib di pelajari :

  • meningkatkan lagi belajar frontend dan backend di tahap akhir, agar bisa jadi fullstack ditahap akhir.
  • melanjutkan belajar bahasa Pemrograman PHP sampai tahap akhir materi.
  • melanjutkan materi react sampe akhir atau sebutuhnya.
  • Mempelajari materi python di tahap akhir sesuai role nya kamu seperti fulstack.
  • Mencoba bekerja magang atau kalo bisa netap di suatu perusahaan cv, pt , dan perkantoran (kalo hoki).
  • Mencoba mempelajari digital marketing ,dll.

Yang disarankan untuk pemula adalah mengambil peran frontend developer agar di ajari dasar-dasar ngoding.
